# Task Management API

This repository contains a Task Management API that provides functionality to create, retrieve, update, and delete tasks. The API is built using Node.js, Express.js, and MySQL.

## Features

- Create a new task
- Get all tasks
- Get a specific task by ID
- Update a task by ID
- Delete a task by ID

## Requirements

- Node.js (version >= 14)
- MySQL database
